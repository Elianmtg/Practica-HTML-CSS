Etiquetas semanticas estructurales

div = Es una etiqueta contenedora que no tiene semantica.
header = La cabecera de un sitio o de una seccion.
main = define la seccion principal de un documento, solo puede existir una etiqueta main por docuemtno.
footer = Pie de pagina de un sitio web o de una seccion.
nav = Representa una navegacion
article = Representa una seccion autocontenido (que por si sola se explica)
aside = Representa un contenido complementario o suplementario.
section = Representa una seccion de contenido generico.
address = Representa una direccion.
